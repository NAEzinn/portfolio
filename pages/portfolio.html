Projeto realizado em python sobre reconhecimento facial. A necessidade desse projeto surgiu ao observar a falta de uso das catracas já instaladas no Senai. O reconhecimento facial se mostrou essencial para o avanço de controle de entrada, permitindo assim o uso adequado das catracas. Isso resulta em uma significativa redução de custos e maior eficiência na supervisão. Este projeto solucionará os problemas de identificação nas catracas, diminuindo os custos para a empresa, além de trazer uma camada extra de segurança para todos os funcionários, pois terá facilidade em reconhecer as pessoas que atuam ou não atuam na empresa.


Projeto realizado em C++ sobre uma ficha de um personagem. O projeto contendo diversos loops e condições para criar a identificação total de um personagem, atribuindo a ele todos seus atributos e características para no final imprimir o resultado na tela de como ficou o personagem.
